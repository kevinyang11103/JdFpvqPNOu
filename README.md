# 项目简介 python64

本项目是一款基于Python和Django框架的豆瓣图书数据可视化分析推荐系统。

## 技术栈

- 后端：Python, Django
- 数据处理：Pandas
- 数据存储：MySQL
- 数据可视化：Echarts
- 推荐算法：协同过滤算法（UserCF）

## 功能模块

- 用户注册与登录
- 个性化图书推荐
- 热门书籍浏览
- 历史行为定制化推荐
- 数据可视化展示

## 系统角色

- 注册用户：可查看推荐书籍，浏览热门书籍，查看个人阅读趋势等

## 运行环境

- Python环境（具体版本号视项目实际需求而定）
- Django框架（具体版本号视项目实际需求而定）
- MySQL数据库（具体版本号视项目实际需求而定）

## 部署步骤

1. 搭建Python环境
2. 安装Django框架
3. 配置MySQL数据库
4. 部署后端服务
5. 前端页面接入
